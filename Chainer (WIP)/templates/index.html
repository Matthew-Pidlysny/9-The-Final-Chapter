The 12 Universal System Rules

Rule 1: Base Relativity

A number's classification (Simple/Wild) depends on the base being used. The same number can be Simple in one base and Wild in another.

Rule 2: Simple Definition

A number is Simple in base b if and only if all its prime factors divide b. This ensures its reciprocal terminates.

Rule 3: Wild Definition

A number is Wild in base b if it has at least one prime factor that does not divide b. This causes its reciprocal to repeat infinitely.

Rule 4: Termination Criterion

The reciprocal 1/n terminates in base b if and only if n is Simple in base b.

Rule 5: The Number 1

The number 1 is always Simple in every base, as it has no prime factors.

Rule 6: Factor Primes

The Factor primes of a base are precisely the prime factors of that base. For base 10: {2, 5}.
