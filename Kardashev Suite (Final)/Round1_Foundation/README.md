## üöÄ Quick Start

### Prerequisites

-   C++17 compatible compiler
-   CMake 3.16 or higher
-   GTK+ 3.0 development libraries
-   OpenGL support

### Building

```bash
# Clone the repository
git clone https://github.com/9Software/KardashevSuite.git
cd KardashevSuite/Round1_Foundation

# Configure and build
mkdir build && cd build
cmake ..
make -j$(nproc)

# Run the application
./kardashev_suite
```

### Industrial Deployment

```bash
# For industrial-grade deployment
cmake -DINDUSTRIAL_DEPLOYMENT=ON -DCMAKE_BUILD_TYPE=RelWithDebInfo ..
make -j$(nproc)

# Install system-wide
sudo make install
```
