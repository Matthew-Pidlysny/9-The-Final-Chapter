### Building All Workshops
```bash
cd KardashevSuite/Round2_Workshops
mkdir build && cd build
cmake -DBUILD_K5_WORKSHOPS=ON -DBUILD_K4_QUANTUM_WORKSHOPS=ON ..
make -j$(nproc)

# Run the integrated workshop system
./kardashev_workshops --max-mode
```
