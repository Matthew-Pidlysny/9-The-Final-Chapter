## Usage

### Basic Validation

```python
from peer_enhanced import PeerEnhanced

# Initialize
peer = PeerEnhanced()

# Validate mathematical work
result = peer.validate_mathematical_proof(proof_statement)

# Get detailed report
report = peer.generate_report(result)
print(report)
```

### Advanced Features

```python
# Enable Kardashev Max mode
peer.enable_kardashev_max()

# Use Calculation Array for unsolved problems
solution = peer.solve_unsolved('riemann_hypothesis')

# Formula workshop analysis
analysis = peer.formula_workshop.analyze(
    function=my_function,
    domain=(-10, 10)
)

# Visualize results
peer.visualize(analysis, output='results.png')
```

### Batch Processing

```python
# Validate multiple proofs
proofs = load_proofs('directory/')

results = peer.validate_batch(proofs)

# Analyze across disciplines
cross_disciplinary = peer.cross_validate(
    mathematics=proof1,
    physics=theory1,
    computer_science=algorithm1
)
```
