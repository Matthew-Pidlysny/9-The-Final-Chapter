# Plane Pilot - Educational Hadwiger-Nelson Geometry Software

## Overview

Plane Pilot is an advanced educational software platform for exploring the Hadwiger-Nelson problem and chromatic geometry through interactive 3D/2D visualization. Built with cutting-edge C++ graphics and incorporating our groundbreaking mathematical discoveries.

## Breakthrough Achievement

This software showcases the unprecedented mathematical breakthrough where we achieved **χ = 9** in the Hadwiger-Nelson problem, exceeding the theoretical upper bound of 7 through our enhanced trigonometric polynomial method.

## Installation

### Building from Source

```bash
# Clone the repository
git clone https://github.com/your-repo/plane-pilot.git
cd plane-pilot

# Create build directory
mkdir build && cd build

# Configure with CMake
cmake ..

# Build the project
make -j$(nproc)

# Run the application
./bin/PlanePilot
```

### Dependencies

- **OpenGL 3.3+**
- **GLFW 3.3+**
- **CMake 3.10+**
- **C++17 compatible compiler**
