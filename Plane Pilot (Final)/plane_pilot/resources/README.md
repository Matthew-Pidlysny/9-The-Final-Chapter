# Plane Pilot Resources

This directory contains all the resources needed for the Plane Pilot educational software.

## Contents

- `splash_image.png` - Breakthrough splash screen image
- `icons/` - Application icons and UI elements
- `fonts/` - Font files for text rendering
- `shaders/` - OpenGL shader files
- `models/` - 3D model files
- `textures/` - Texture files for 3D rendering

## Adding Resources

To add new resources, place them in the appropriate subdirectory and update the CMakeLists.txt file to include them in the build.
