# Plane Pilot Educational Software

## Overview
Plane Pilot is a comprehensive C++ educational framework for exploring advanced mathematical concepts, specifically the Hadwiger-Nelson problem, chromatic theory, and 3D geometric modeling.

## Features

### Four Main Workshops:

1. **Essential Hadwiger-Nelson** 
   - Comprehensive 3D and 2D visualization of the Hadwiger-Nelson problem
   - Interactive chromatic graph analysis
   - Real-time unit-distance edge detection
   - Mathematical proof visualization
   - Breakthrough solution demonstration

2. **Chromatic Olympics**
   - Interactive chromatic number challenges
   - Competitive graph coloring games
   - Educational geometry competitions

3. **Geometry of Proper Chromatic Sequencing**
   - Advanced geometric sequence analysis
   - Proper chromatic arrangement studies
   - Mathematical pattern recognition

4. **Sphere-O-Matic**
   - 3D sphere modeling and AI-generated geometry
   - Integration with je-suis-ballin.py sphere models
   - Collision detection at extremal configurations
   - AI-powered novel geometry generation
   - Real-time physics simulation

## System Requirements
- C++17 compatible compiler
- OpenGL 3.3 or higher
- GLFW3
- GLU (OpenGL Utility Library)
- Linux/Unix system

## Installation and Usage

### Build from Source:
```bash
cd plane_pilot
chmod +x build.sh
./build.sh
```

### Run the Application:
```bash
cd dist
./bin/PlanePilot
```

## Package Contents
- `dist/bin/PlanePilot` - Main executable
