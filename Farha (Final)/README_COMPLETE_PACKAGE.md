# Farha Educational Game - Complete Package

## Bismillah ir-Rahman ir-Raheem

**Farha**: Child-friendly educational game about establishing the Rashiddun Caliphate

### Package Contents:

#### Version 1: Basic Educational Experience
- **Location**: Version1_Basic/
- **Features**: Basic game mechanics, simple educational content
- **Build**: cd Version1_Basic && ./build_farha.sh
- **Run**: cd Version1_Basic/build && ./farha_game

#### Version 2: Enhanced Educational Experience  
- **Location**: Version2_Enhanced/
- **Features**: Enhanced gameplay, additional educational features
- **Build**: cd Version2_Enhanced && ./build_enhanced.sh
- **Run**: cd Version2_Enhanced/build_enhanced && ./farha_enhanced

#### Version 3: Allah Inspired Educational Experience
- **Location**: Version3_AllahInspired/
- **Features**: 5000+ Allah-inspired elements, comprehensive Islamic education
- **Build**: cd Version3_AllahInspired && ./build_allah_inspired.sh
- **Run**: cd Version3_AllahInspired/build_allah_inspired && ./farha_v3

### Special Features of Version 3:
- **500 Qur'anic Verses**: Direct verses and educational applications

- **500 Islamic Values**: Character building and moral education

### Building Instructions:
1. Extract this package to your desired location
2. Navigate to any version directory
3. Run the build script (build_farha.sh, build_enhanced.sh, or build_allah_inspired.sh)
4. Run the executable from the build directory

### System Requirements:
- Linux/Unix system with C++ compiler
- CMake (version 3.10 or higher)
- Basic terminal knowledge
