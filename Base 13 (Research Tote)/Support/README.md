# Base 13 Research Project - Complete Package

This package contains comprehensive research documentation and testing tools for three interconnected mathematical systems centered around the number 13.

## Contents

### 1. Research Documents (LaTeX)

#### BASE13_COMPREHENSIVE_STUDY.tex
A comprehensive 39+ page study of the Base 13 (tridecimal) mathematical system covering:
- Mathematical foundations and properties
- Divisibility rules and modular arithmetic
- Fraction representation and periodicity
- The Plus 3 Phenomenon
- Computational applications
- Connections to other mathematical structures
- Pedagogical considerations

#### SEQUINOR_TREDECIM_COMPREHENSIVE.tex
A comprehensive 39+ page study of the Sequinor Tredecim system covering:
- The 13 fundamental components (Alpha through Nu)
- Mathematical definitions and properties
- Component interactions and integration
- Applications in cryptography, calendars, and signal processing
- Computational implementation
- Theoretical extensions

#### REMAINDER_SYSTEM_COMPREHENSIVE.tex
A comprehensive 39+ page study of the remainder-based counting system covering:
- Remainder-based organization of integers
- The Plus 3 Phenomenon in detail
- Scaling analysis across multiple orders of magnitude
- Computational analysis and verification
- Connections to number theory
- Generalizations and extensions

### 2. Testing Suite (Python)

Interactive workshop for Base 13 fundamentals including:
- Decimal to Base 13 conversion tests

#### workshop_module_2_sequinor.py
Interactive workshop for Sequinor Tredecim exploration including:
- Alpha component (scaling) tests

- Gamma component (periodicity) tests
- Delta component (deviation) tests

Interactive workshop for integrated systems analysis including:

Main test runner that executes all three workshop modules and provides comprehensive summary.

## Usage

### Compiling LaTeX Documents

To compile any of the LaTeX documents:

```bash
pdflatex BASE13_COMPREHENSIVE_STUDY.tex
pdflatex SEQUINOR_TREDECIM_COMPREHENSIVE.tex
pdflatex REMAINDER_SYSTEM_COMPREHENSIVE.tex
```

For complete compilation with table of contents and references:
```bash
pdflatex document.tex
pdflatex document.tex
```

### Running the Testing Suite

To run all workshop modules:
```bash
python run_all_workshops.py
```

To run individual workshops:
```bash
python workshop_module_1_base13.py
python workshop_module_2_sequinor.py
python workshop_module_3_integrated.py
```

### Running Analysis Scripts

```bash
python base13_remainder_calculator.py
python sequinor_verification.py
python pidlysnian_validation_system.py
```
